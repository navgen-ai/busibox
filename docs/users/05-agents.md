# AI Agents

AI agents are assistants that can search your documents, browse the web, and answer questions. They're the main way you interact with your content through conversation.

## What Agents Are

Agents are AI-powered assistants with different capabilities:

- **Document search** — They can search your authorized documents and use them to answer questions with citations.
- **Web search** — They can look up information on the internet for questions about external topics.
- **Tools** — They may have access to other tools (calculations, lookups, etc.) depending on how your admin configured them.

You chat with agents the same way you'd chat with a colleague — type a question, get a response. The difference is they can instantly search thousands of documents and synthesize answers.

## The Chat Interface

When you open an agent (for example, in Agent Manager or the AI Portal chat):

- **Message box** — Type your question and press Enter or click Send.
- **Streaming responses** — Answers appear word by word as they're generated. You don't have to wait for the full response.
- **Citations** — When the agent uses your documents, it shows where the information came from. Click a citation to jump to the source.

You can have back-and-forth conversations. The agent remembers the context of the current chat.

## Document Search Agent

The document search agent is built for questions about your content. When you enable document search:

- Ask things like "What are the main risks in the project plan?" or "Summarize the Q3 financial results."
- The agent searches your documents, finds relevant passages, and answers using that information.
- It cites the documents and passages it used.

This is ideal for research, summarization, and finding specific information across many files.

## Web Search Agent

For questions that aren't in your documents — current events, general knowledge, external data — the web search agent can look things up online. It will search the web and incorporate results into its answer.

Your admin controls whether web search is enabled and for which agents.

## Attaching Files to Conversations

Some agents let you attach files directly to a message. The agent can then use that file in addition to (or instead of) searching your document library. Useful when you have a one-off document or want to focus on a specific file.

## Conversation History

Conversations are saved. You can:

- **Resume** — Come back to a previous conversation and continue where you left off.
- **Review** — Scroll through past messages and citations.
- **Start fresh** — Begin a new conversation when you want a clean slate.

Your admin may set retention policies for how long conversations are kept.

## Tips for Getting Good Answers

- **Be specific** — "What are the budget assumptions in the Q3 forecast?" works better than "Tell me about the budget."
- **Enable document search** — For questions about your content, make sure document search is on. Otherwise the agent may answer from general knowledge only.
- **Provide context** — If the agent seems off, add context. "In the project plan document, what are the milestones?"
- **Check citations** — When the agent cites a document, click through to verify. Citations help you trust and extend the answer.
